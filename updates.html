First Project
January 2nd, 2024

Today, I found the motivation to start the first game that will be released on this website. It is a first person shooter where you are on a planet far out in space with either aliens or other players as enemies(I haven't decided which would be cooler). Anyway, I have unique ideas for the weapons and damage system that I am excited to try and add to this game.

I started progress late in the day, but got some good work done in a few hours. Currently, there is a pill with a brick that can shoot balls. While it may not sound like much, the movement and shooting systems are some of the most difficult and key parts of this game, so it's a good way to start. I'll be able to provide images and videos of this project once I get a few more things done.

As for the website, get used to how it is right now. I plan to update it once I'm done with this project. Things I'm currently looking at adding include and easier way for me to edit this updates page(I have to edit the file every time), and a newsletter system.
